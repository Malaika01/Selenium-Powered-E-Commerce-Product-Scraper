# 🛒 Selenium-Powered E-Commerce Product Scraper

A Python-based web scraping project that uses Selenium to extract product information from an e-commerce website (e.g., Amazon). The scraper collects product details such as title, ratings, price, and image URL for multiple search queries.

## 📌 Features

Automated web scraping using Selenium

Extracts product details:

Product Title

Ratings

Price

Image URL

Supports multiple search queries

Scrapes multiple pages per query

Saves results into structured files

## ⚙️ Dependencies

Python 3.x

Selenium

WebDriver Manager

Install dependencies:

'''pip install selenium webdriver-manager'''

## ▶️ How to Run

Clone the repository:
```git clone https://github.com/Malaika01/Selenium-Powered-E-Commerce-Product-Scraper.git```

```cd Selenium-Powered-E-Commerce-Product-Scraper```


Run the script:

```python main.py```
